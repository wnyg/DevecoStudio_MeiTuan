# 更新日志 (Changelog)

本项目的所有重要变更都将记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.2.0] - 2023-10-25

### 新增
- 新增了用户头像上传功能。
- 增加了暗黑模式切换按钮。

### 修复
- 修复了在 iOS 设备上输入框被键盘遮挡的问题。
- 修复了登录页面在弱网环境下偶尔崩溃的 Bug。

### 变更
- 将 API 请求的超时时间从 5 秒延长至 10 秒。

---

## [1.1.0] - 2023-09-15

### 新增
- 支持 GitHub 第三方登录。

### 移除
- 移除了过时的 `v1/legacy` 接口。

---

## [1.0.0] - 2023-08-01

### 新增
- 项目正式发布，包含基础的用户注册与登录功能。
